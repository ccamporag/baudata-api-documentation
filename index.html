/permisos

Retrieve information about PE in a time window

GET /permisos?fecha_ingreso_desde=YYYY-MM-DD&fecha_ingreso_hasta=YYYY-MM-DD&fecha_permiso_desde=YYYY-MM-DD&fecha_permiso_hasta=YYYY-MM-DD

Params:


fecha_ingreso_desde: Date of entry into the baudata database (from) in YYYY-MM-DD format. Default is 2020-01-01.

fecha_ingreso_hasta: Date of entry into the baudata database (up to) in YYYY-MM-DD format. Default is today.

fecha_permiso_desde: Building permit date (from) in YYYY-MM-DD format. Default is 60 days ago.

fecha_permiso_hasta: Building permit date (up to) in YYYY-MM-DD format. Default is today.


Python Example:

import requests
APITOKEN = 'MY-API-TOKEN'
params = {"fecha_permiso_desde": "2023-07-01", "fecha_permiso_hasta": "2023-07-15"}
headers = {'Authorization': APITOKEN}
response = requests.get("https://rf2d2rjlla.execute-api.sa-east-1.amazonaws.com/permisos", params=params, headers = headers)
print(response.json())

/constructoras

Retrieve information about contractors

GET /constructoras?n=N

Params:


n: Number of contractors to retrieve. The contractors are ordered by square meters of building permits, for example, n = 10 will return the top 10 contractors in the database. Default is n = 100000


Python Example:

import requests
APITOKEN = 'MY-API-TOKEN'
params = {"n": 10}
headers = {'Authorization': APITOKEN}
response = requests.get("https://rf2d2rjlla.execute-api.sa-east-1.amazonaws.com/constructoras", params=params, headers = headers)
print(response.json())
