/permisos

Retrieve information about PE in a time window

GET /permisos?fecha_ingreso_desde=YYYY-MM-DD&fecha_ingreso_hasta=YYYY-MM-DD&fecha_permiso_desde=YYYY-MM-DD&fecha_permiso_hasta=YYYY-MM-DD&fecha_actualizado_desde=YYYY-MM-DD&fecha_actualizado_hasta=YYYY-MM-DD&superficie_desde=0&superficie_hasta=10000000

Params:


fecha_ingreso_desde: Date of entry into the baudata database (from) in YYYY-MM-DD format. Default is 2020-01-01.

fecha_ingreso_hasta: Date of entry into the baudata database (up to) in YYYY-MM-DD format. Default is today.

fecha_permiso_desde: Building permit date (from) in YYYY-MM-DD format. Default is 60 days ago.

fecha_permiso_hasta: Building permit date (up to) in YYYY-MM-DD format. Default is today.

fecha_actualizado_desde: Last update of the record in YYYY-MM-DD format (from). Default is 2020-01-01.

fecha_actualizado_hasta: Last update of the record in YYYY-MM-DD format (up to). Default is today.

superficie_desde: PE square meters (from). Default is 0 m2

superficie_hasta: PE square meters (up to). Default is 10.000.000 m2


Python Example:

import requests
APITOKEN = 'MY-API-TOKEN'
params = {"fecha_permiso_desde": "2023-07-01", "fecha_permiso_hasta": "2023-07-15", "superficie_desde: 1000"}
headers = {'Authorization': APITOKEN}
response = requests.get("https://rf2d2rjlla.execute-api.sa-east-1.amazonaws.com/permisos", params=params, headers = headers)
print(response.json())
